# Nextik

Full-stack web application built with Next.js demonstrating modern authentication flows and full-stack architecture.

The project currently includes user authentication and serves as a foundation for further development (CRUD features, dashboard, and additional functionality).

## Features

- User registration and login
- Credentials-based authentication
- OAuth authentication (Google, GitHub)
- JWT-based session handling
- Protected authentication routes
- Form validation

## Tech Stack

Frontend
- React
- Next.js
- TypeScript

Backend
- Next.js API routes
- NextAuth

Database
- Prisma ORM
- SQLite

Validation
- Zod
